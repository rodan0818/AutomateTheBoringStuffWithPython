- Functions are like a mini-program inside your program.
- The main point of functions is to get rid of duplicate code.
- The def statement defines a functions
- The input to functions are arguments. The output is the return value.
- The parameters are the varibles in betwwn the function's paretheses
  in the def statement.
- The return value is specified using the return statement.
- Every funtion has a return value. If your function doesn't have a return statement, the default return value is None.
- Keyword arguments to functions are usually for optional arguments.
- The print() function has keyword arguments end and sep
  end argument replaces the print fucntion's default newline print
  sep argument replaves the print function's default single space print
