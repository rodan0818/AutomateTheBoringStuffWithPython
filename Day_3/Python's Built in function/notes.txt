From now we will have a notes.txt included with every folder as like this

- You can import modules and get access to new functions.
- The modules that come with python are called the standard library,
  but you can also install third-party modules using the pip tool.
- The sys.exit() function will immediately quit your program.
- The pyperclip third-party module has copy() and paste() functions
  for reading and writing text to the clipboard
